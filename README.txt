== Installation ==

= INSTALL WPFPDF FROM WITHIN WORDPRESS =

1. Visit the plugins page within your dashboard and select ‘Add New’;
1. Search for ‘PDF Generator Add-on for WPForms;
1. Activate ‘PDF Generator Add-on for WPForms from your Plugins page;

= INSTALL WPFPDF MANUALLY =

1. Upload the ‘bvnode-wpf-pdf-generator’ folder to the /wp-content/plugins/ directory;
1. Activate the ‘PDF Generator Add-on for WPForms plugin through the ‘Plugins’ menu in WordPress;
